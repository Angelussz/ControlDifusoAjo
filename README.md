  - Ideal: 18°C - 24°C
  - Alta: 24°C - 30°C

  - Ideal: 13°C - 18°C
  - Alta: 18°C - 30°C
- Maduración
  - Bajo: 0°C - 20°C
  - Ideal: 20°C - 25°C
  - Alta: 25°C - 30°C

- Alta: 0 (60°C - 100°C)
